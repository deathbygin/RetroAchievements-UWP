# RetroAchievements-UWP

A vibe-coded UWP front-end for the Retroachievements API.
Made for myself as a quick way to check on achievements without having to deal with the clunky edge UI on XBOX.

##Instructions

1. Retrieve your API key from https://retroachievements.org/settings.
2. Enter your AIP key and username.
3. Enjoy!
